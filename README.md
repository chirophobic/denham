# Denham

> A vending machine and warehouse application using DDD, TDD, and BDD.

## Installation

This project requires the following tools:

1. NodeJS
2. Yarn
3. Pulumi

## Getting started

```bash
# build the whole application
yarn build:all

# run all the tests
yarn test:all

# lint all source code
yarn lint:all
```
